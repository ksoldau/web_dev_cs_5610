Some experiments with jQuery interactions

Source

HTML CSS JS/jQuery

Purpose

To figure out how to use some jQuery interactions such as draggable, droppable, resizable, and selectable.

Experiments

Draggable

Drag me!

HTML:
<div class="draggable"> <p>Drag me!</p> </div>
jQuery:
$(".draggable").draggable();

Droppable

Drag me onto the red shape!

HTML:
<div class="draggable"> <p>Drag me onto the red shape!</p> </div> <div id="droppable" class="muted-red"> <p id="drop-p"></p> </div>
jQuery:
$(".draggable").draggable(); $("#droppable").droppable({ drop: function(event, ui) { $(this) .addClass("pure-red") .find("#drop-p") .html("Dropped!"); } });

Resizable

I'm resizable!

HTML:
<div id="resizable-square"> <p> I'm resizable! </p> </div>
jQuery:
$("#resizable-square").resizable();

Selectable

  • You Belong with Me
  • Brown Eyed Girl
  • Breakaway
  • Life's What You Make It
  • Who Says
  • Daughters
  • Bring it All Back
HTML:
<ul id="selectable"> <li>You Belong with Me</li> <li>Brown Eyed Girl</li> <li>Breakaway</li> <li>Life's What You Make It</li> <li>Who Says</li> <li>Daughters</li> <li>Bring it All Back</li> </ul>
jQuery:
$("#selectable").selectable();

Explanations

To do these experiments I looked at the interactions listed under the interactions label on their website of the jQuery UI site.. For all of the experiments except for droppable I just had to apply the necessary jQuery function to the correct DOM element. For droppable I essentially did the same thing, I just also had to code what should happen when the drop occurred as an argument into the .droppable() function.

Bibliography

The jQuery UI page on .draggable() was used as reference: http://jqueryui.com/draggable/. The page on .droppable() was heavily referenced: http://jqueryui.com/droppable/. The page on .resizable() was referenced: http://jqueryui.com/resizable/. And, finally, the page on .selectable() was referenced and the CSS was taken from it for my own use: http://jqueryui.com/selectable/.
