Title: Changing element's style with jQuery

Source

HTML CSS JS/jQuery

Purpose

To change an element's style using jQuery.I changed margins, background-colors, fonts, font-color, width, height, and position. Also to figure out how to select elements by tag type, class, and id.

Experiments

Changing an object's margin

HTML:
<div id="rectangle"></div>
jQuery:
$("#rectangle").css("margin-left", "200px");

Changing object's background color

HTML:
<div class="box"></div> <div class="box"></div>
jQuery:
$(".box").css("background-color", "#999");

Changing an object's font

This is font that is styled to be a serif font in the stylesheets, but is changed to a sans-serif font with jQuery.

HTML:
<p id="paragraph">This is font that is styled to be a serif font in the stylesheets, but is changed to a sans-serif font with jQuery.</p>
jQuery:
$("#paragraph").css("font-family", "sans-serif");

Changing an object's font color

This object's font color is #999 in the stylesheet, but set to #900 with jQuery

HTML:
<p id="sentence">This object's font color is #999 in the stylesheet, but set to #900 with jQuery</p>
jQuery:
$("#sentence").css("color", "#900");

Changing an object's width and height

HTML:
<div id="rectangle2"></div>
jQuery:
$("#rectangle2").css("width", "200px").css("height", "200px");

Changing an element's left position when it's absolutely positioned

HTML:
<div id="flowers-container"> <img id="flowers" src="http://icons.mysitemyway.com/wp-content/gallery/glossy-black-icons-natural-wonders/thumbs/thumbs_051369-glossy-black-icon-natural-wonders-flower28-sc44.png"></img> </div>
jQuery:
$("img").css("left", "300px");

Explanation

To change an element's style with jQuery I got ahold of the element by the element's class, id, or tag then applied the css function to it. To alter the style I then added the css property in a string as the first argument to the css function and the value as the second argument.

For example, to alter the background-color of elements with the class of "box" I used this jQuery:
$(".box").css("background-color", "#999");

The above code grabs the elements with the class of box by $(".box") then applies the css function to it. The first argument to the css function is a string of the property I wanted to alter and the second argument is a string that's the new value I want to give said property.

This process was used for every mini-experiment shown on this page. The only differences in the jQuery was that a different class, id, or tag was grabbed, a different property was given as the first argument to css, and the desired value for said property was given for the second argument to css.
