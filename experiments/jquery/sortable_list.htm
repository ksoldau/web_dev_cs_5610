Title: Sortable lists

Source

HTML CSS JS/jQuery

Purpose

The purpose of this experiment was to figure out how to make sorted lists. I did two types of sorted lists: one where you could move the pieces anywhere (movement not restricted by axis) and one where you could only move the elements in the list up and down.This could be useful if I implement the ability for a user to list their favorite tabs/chord listings.

Explanation

To implement these sorted lists I first created unordered lists. I filled in the unordered lists with lis that were song names. In order to make the lists sortable I then applied the jQuery .sortable() function to each. To make the sorted list whose items could only move up and down I added in the argument {"axis" : "y"} to the .sortable() function in order to restrict movement to the y-axis. The styling of the lists was mostly taken from the jQuery UI website.

Bibliography

The CSS of the lists was taken from http://jqueryui.com/sortable/. I just altered the background-colors a bit. I also used that page as reference to figure out what kind of HTML and jQuery was needed to create sorted lists.
