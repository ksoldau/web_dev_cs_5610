Title: Highlight all divs up to the one clicked on

Source:

HTML CSS JS/jQuery

Purpose

The purpse of this was to find out how to add and remove classes to various divs based on a user's clicks. This could be useful on my site to rate chord listings/tabs

Experiment

clear rating
HTML:
<div id="flowers-container"> <img id="flowers" src="http://icons.mysitemyway.com/wp-content/gallery/glossy-black-icons-natural-wonders/thumbs/thumbs_051369-glossy-black-icon-natural-wonders-flower28-sc44.png"></img> </div>
$(".square").click(function() { if (!($(this).hasClass("selected"))) { $(this).addClass("selected"); $(this).prevAll().addClass("selected"); } else { $(this).nextAll().removeClass("selected"); } }); $(".clear-ratings").click(function() { $(this).prevAll().removeClass("selected"); });

Explanation

To make the rating system I first made five divs that all had the same class of 'square'. Using jQuery I then grabbed all of those divs and assigned what would happen if any of them were clicked on by using the '.click()' function. If a square is clicked on that does not currently have the class 'selected' applied to it then that square has the class 'selected' applied to it by the .addClass() method and it's made sure that all of the divs before it are also given the 'selected' class by using the .nextAll() and .addClass() methods. If the square that's clicked on already has the 'selected' class applied to it then that square and all of the proceeding squares have the class "selected" removed. That was done using the .prevAll() and .removeClass() functions in jQuery. All of this makes it so that, no matter what square div is clicked on, the squares that appear 'selected' are only the stars preceding and including the last star clicked on. The clear rating button works to clear the 'select' class off of all of the previous divs using the .prevAll() and .removeClass() methods.
