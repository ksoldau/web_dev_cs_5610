Dialog that opens on click event

This is the dialog that comes up automatically

Click here to open a dialog

You clicked on something to get this

HTML:
<div id="dialog" title="Are you sure?"> <p> This is the dialog that comes up automatically </p> </div>
jQuery:
$("#click-dialog").dialog({autoOpen: false}); $(".click-here").click(function() { $("#click-dialog").dialog('open'); });

Explanations

To learn how to do these experiments I looked at the widgets on the jQuery UI page (http://jqueryui.com/. To make the buttons I grabbed the DOM elements I wanted then applied the .button() function to them. To make a dialog automatically pop-up I grabbed the dialog by its id then applied the .dialog() function to it. To create a dialog that opened on a click event I grabbed the DOM elements that would be the dialog, applied the .dialog() function to them, then grabbed the element I wanted to be clicked on and made the dialog open on a click event using the .click() and .dialog() functions. To make the dialog open you just pass 'open' in as an argument to .dialog(). To make the tooltip I just created an input that had what I wanted the tooltip to be as the title.
