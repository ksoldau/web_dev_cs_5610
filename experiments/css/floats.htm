How to clear floats

Why do floats need to be cleared?

Sometimes containers don't expand to fit it. This can result in layout that you want to be underneath a floated object to creep up next to it.

How to clear a float: put the following CSS in the float's container:
.float-container { overflow: auto; width: 100%; }

or add the CSS clear: both to a tag so it clears the floats

No float styling

Below is what happens when you don't float either square. (The default float value is 'none.')

the HTML:
<div class="square blue"></div>
<div class="square pink"></div>"

the CSS:
.square { width: 100px; height: 100px; }
.blue { background-color: #4444FF; }
.pink { background-color: #FF70A8; }

Float right

Below is what happens when you float the blue square to the right and do not float the pink square and the blue square is coded on top of the pink square. Notice that the blue square does not stay above the pink square. Since the blue square is floated to the right (Since the pink square is not floated, it's float value is the default value 'none.'

the HTML:
<div class="square blue float-right"></div>
<div class="square pink"></div>

the CSS:
.square { width: 100px; height: 100px; }
.blue { background-color: #4444FF; }
.pink { background-color: #FF70A8; }
.float-right { float: right; }

Below: Two divs. First div has default float, second is floated to the right. Notice that the blue square stays above the pink square. Floats needed to be cleared after this one.

the HTML:
<div class="square blue"></div>
<div class="square pink float-right"></div>

the CSS:
.square { width: 100px; height: 100px; }
.blue { background-color: #4444FF; }
.pink { background-color: #FF70A8; }
.float-right { float: right; }

Below: both squares floated to the right.

the HTML:
<div class="square blue float-right"></div>
<div class="square pink float-right"></div>

the CSS:
.square { width: 100px; height: 100px; }
.blue { background-color: #4444FF; }
.pink { background-color: #FF70A8; }
.float-right { float: right; }

Float left

Blue floated to the left, pink with default float. Pink ends up beneath the blue one.

the HTML:
<div class="square blue float-left"></div>
<div class="square pink"></div>

the CSS:
.square { width: 100px; height: 100px; }
.blue { background-color: #4444FF; }
.pink { background-color: #FF70A8; }
.float-left{ float: left; }

Pink floated to the left, blue with default float.

the HTML:

<div class="square pink float-left"></div>

the CSS:
.square { width: 100px; height: 100px; }
.blue { background-color: #4444FF; }
.pink { background-color: #FF70A8; }
.float-left{ float: left; }

Both squares floated to the left. Blue goes as far as it can go to the left, then pink goes as far as it can go.

the HTML:

the CSS:
